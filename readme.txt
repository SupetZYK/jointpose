This package contains the following elements:

core - pose estimation code of our CVPR 16 paper
configs_object_pose_estimation - configuration files containing the parameters used in the object pose estimation experiments of the CVPR 16 paper (RGB and RGB-D)
configs_camera_localization - configuration files containing the parameters used in the camera localization experiments of the CVPR 16 paper (RGB and RGB-D)

readme.txt - this file
license.txt - license information for the code
documentation.pdf - document explaining compilation of and running the code, data structure and parameter options
